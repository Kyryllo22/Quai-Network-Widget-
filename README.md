Install the Quai Network Widget on iPhone (via Scriptable)

1️⃣ Download the Scriptable app from the App Store:  
https://apps.apple.com/ru/app/scriptable/id1405459188  

Open the app once after installing.

2️⃣ Add the widget script  
1. Open the script and copy it.  
2. Go to Scriptable, add the script to the app, name it "QuaiWidget", and save it.

3️⃣ Add the widget to your Home Screen  
1. Press and hold your iPhone home screen until the icons start shaking.  
2. Tap + in the top left corner.  
3. Search for Scriptable.  
4. Choose the Large widget and tap Add Widget.  
5. Tap the widget → select your script (QuaiWidget).  
6. Done.

✅ The widget will start showing Quai Network data (SOAP revenue, market metrics, etc.).  
It may take a few minutes to update.
